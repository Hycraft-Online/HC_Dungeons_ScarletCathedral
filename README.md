# HC_Dungeons_ScarletCathedral

A WoW-inspired dungeon instance plugin for Hytale implementing the Scarlet Monastery: Cathedral wing. Enforces dungeon rules by preventing block breaking and placement within dungeon instance worlds.

## Features

- Cancels all block break events inside Scarlet Monastery instance worlds
- Cancels all block placement events inside Scarlet Monastery instance worlds
- Scoped to worlds with the `instance-ScarletMonastery` name prefix, leaving other worlds unaffected
- Includes asset pack for dungeon content

## Dependencies

None.

## Building

```bash
./gradlew build
```
